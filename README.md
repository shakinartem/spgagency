# SPG Static Site

## Состояние проекта

### Уже есть

- Основной одностраничный сайт SPG на `React + TypeScript + Vite + Tailwind CSS + Framer Motion`.
- Отдельная страница `materialy.html` с архивом материалов.
- Набор SEO-страниц под ниши и статьи.
- Форма аудита с отправкой в Telegram через webhook и fallback-сценарием.
- Статические страницы `privacy.html` и `terms.html`.
- Деплой под GitHub Pages с учетом `base path`.

## Быстрый запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
```

Предпросмотр:

```bash
npm run preview
```
